## 9. ChatGPT 앱 활용

### ChatGPT 데스크탑 프로그램

ChatGPT 데스크탑 앱은 브라우저 없이 독립적으로 실행되는 네이티브 애플리케이션이다.

- **독립 실행형 앱**: 브라우저 없이 실행 가능하여 더 쾌적한 사용자 경험 제공
- **빠른 접근성과 멀티태스킹 지원**: Alt+Tab 전환으로 업무 효율성 향상
- **네이티브 기능 통합**: 단축키, 다크 모드, 알림 기능 등 윈도우 시스템과 연동
- **성능 최적화**: 브라우저 대비 메모리 사용 감소 및 안정적인 실행

### ChatGPT 앱 설치 방법

다음 경로에서 다운로드 및 설치할 수 있다.

- https://openai.com/chatgpt/download/
- https://apps.microsoft.com/

검색창에 `"chatgpt app download"`를 입력하여 검색하거나, Microsoft Store에서 **ChatGPT Installer.exe**를 다운로드하여 설치한다.

### ChatGPT 앱 실행하기

1. 앱 실행 후 **로그인** 화면에서 이메일 주소를 입력하거나 Google 계정으로 로그인
2. 워크스페이스 선택 화면에서 개인 계정 또는 조직 워크스페이스를 선택
3. 웹 버전과 유사한 화면이 표시되며, 좌측 사이드바에서 이전 대화 및 프로젝트에 접근 가능

## 10. GPT 탐색: 나만의 GPT 활용

### 개인 맞춤형 GPT 생성 개요

**맞춤형 GPT**는 사용자의 필요와 목적에 맞춰 개인화된 GPT 모델이다. 별도의 프로그래밍 지식 없이도 누구나 손쉽게 만들 수 있으며, OpenAI가 **ChatGPT Pro(유료)** 사용자에게 제공하는 기능이다.

**맞춤형 GPT의 특징**
- 사용자가 원하는 목적과 스타일에 따라 GPT의 성격, 행동 및 프롬프트 지침을 자유롭게 설정
- 개인 비서, 학습 튜터, 고객 상담 챗봇 등 다양한 역할을 부여 가능
- 대화 톤, 응답 길이, 주제 범위를 조정할 수 있는 유연성을 제공
- 일상생활이나 업무에서의 생산성을 높이고 AI의 잠재력을 최대한 활용

**맞춤형 GPT 활용 유형 예시**
- **학생용 GPT**: 특정 과목에 대한 질문에 답변하거나 학습 자료를 제공하는 튜터 역할
- **직장용 GPT**: 고객의 문의에 신속하게 대응하는 상담 챗봇
- **주제 범위 조정**: AI의 지식 범위를 확장하여 특정 도메인에 집중
- **응답 길이 조정**: 사용자 요구에 맞춘 AI 응답의 길이 설정
- **대화 톤 조정**: 사용자 선호에 맞는 AI의 말투 설정

### '내 GPT' 만들기 기초

**접근 방법**
1. 오른쪽 상단의 프로필 아이콘을 클릭하여 나타나는 메뉴에서 **'내 GPT'** 선택
2. 또는 **'GPT 탐색하기'** 화면 우측 상단의 **'+ 만들기'** 버튼 클릭
3. 왼쪽 상단의 **'만들기'** 버튼을 클릭하면 GPT의 역할과 목적을 설명하는 단계로 진입

**GPT 만들기 화면 구성**
- **왼쪽 패널**: 구성 (이름, 설명, 지침, 대화 스타터, 기능 설정)
- **오른쪽 패널**: 지금 만든 GPT의 미리보기 및 테스트

**'만들기' 탭 작성 예시**
> "무엇이든 물어보세요" 란에 GPT가 어떤 전문성을 가지고 있는지 명확히 설명한다.
> 예: "당신은 대학생들을 위한 PowerPoint 발표 자료를 만드는 데 특화된 전문 컴퓨터 과학 튜터입니다"

### 내 GPT 'SlidePro for Computer Science' 제작하기

**구성 요소 전체 정리**

| 구분 | 항목 | 작성 내용 |
|---|---|---|
| 만들기 | 무엇이든 물어보세요 | You are a professional computer science tutor who specializes in creating PowerPoint presentation content for undergraduate students. |
| 구성 | 이름 | SlidePro for Computer Science |
| 구성 | 설명 | A GPT for generating technical PowerPoint content for computer science majors, including code samples and explanations. 컴퓨터 과학 전공 대학생을 위한 기술적 PowerPoint 콘텐츠를 생성하는 GPT입니다. |
| 구성 | 지침1 | For every slide, follow this structure: 1. Slide Title / 2. 3-5 subtitles or subtopics below the Slide Title / 3. After each subtitle, write 1-2 explanatory paragraphs of 200-300 characters / 4. Python Code Block / 5. Code Explanation (what the code does, line-by-line if necessary) |
| 구성 | 대화 스타터 | Make 10 detailed slides explaining convolutional neural networks. / Create a presentation on Dijkstra's algorithm with Python code. / Help me generate lecture slides about gradient descent. |
| 구성 | 지식 | GPT에게 참조할 문서, 교재, PDF 파일 등 외부 지식을 업로드하여 그 내용을 바탕으로 답변을 하게 만드는 기능 |
| 구성 | 기능 | 웹 검색, 캔버스, DALL-E 이미지 생성, 코드 인터프리터 및 데이터 분석 |
| 구성 | 작업 | GPT가 외부 데이터를 기반으로 반복 작업이나 고급 분석을 수행할 수 있도록 설정하는 기능 |

**'내 GPT' 테스트**

'미리보기'에서 출시 전에 GPT를 점검한다. 예를 들어 "파이썬 언어 개요의 슬라이드 자료 작성해 줘"라는 질문을 통해 각 슬라이드가 제목, 소주제, 설명, 코드 예제, 코드 설명 구조로 정상 생성되는지 확인한다.

### 내 GPT 'SlidePro for Computer Science' 수정 및 공유하기

**지침 수정 내용**
- 설명 구문을 명사형으로 종료
- 답변의 마지막에 PowerPoint 또는 Jupyter Notebook 파일로 저장할 수 있는 기능을 포함

수정된 지침 예시 (파일 형식 선택 안내 추가):
```
At the end of the response, please include a note that allows the user to save the content as a file.

If needed, please select the format to create the above content as a PowerPoint file or Jupyter Notebook file, or request another format.

file_format
Please select the desired file format by copying the options below and marking with an [x]:
- [ ] PDF
- [ ] PowerPoint (PPTX)
- [ ] Jupyter Notebook (ipynb)
```

**PDF 파일에서 한글 지원**

ChatGPT에서 한글이 포함된 PDF 파일을 만들 때 글꼴 문제로 인해 한글이 제대로 보이지 않을 수 있다. 해결 방법은 다음과 같다.
- '내 GPT' 지침에 한글 폰트 사용 안내 추가: `"When generating a PDF file that contains Korean text, use the attached Korean font"`
- 한글 폰트 파일(예: NanumGothic 등)을 지식(Knowledge)에 업로드

**'내 GPT' 공유 방법**
1. 화면 상단 오른쪽 **[만들기]** 버튼을 눌러 프로젝트 저장
2. 왼쪽 공유 메뉴에서 공개 범위 선택: **'GPT 스토어'** 또는 **'초대된 사람만'** 등
3. 공유할 사람의 이메일 주소를 입력하고 액세스 항목에서 GPT 프로젝트를 확인
4. 오른쪽 아래의 **[업데이트]** 버튼을 눌러 설정 저장 → 공유 완료

**프로필 이미지 등록**

GPT를 만들 때 왼쪽 상단의 '+' 버튼을 눌러 프로필 이미지를 등록하면 GPT의 대표 아이콘이 생성되어 알아보기 쉬워지고, 나만의 GPT에 개성을 부여할 수 있다.
